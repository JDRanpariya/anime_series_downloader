This program downloads the entire series from gogoanime

it requires you to enter the anime name and number of episodes as in the website url
the number of episodes you enter should be less than the length of the anime

Note:
	* The download always starts from the first episode 
	* Make sure that there is no .crdownload file in your downloads folder
	* The speed of the download depends on your internet speed and the server speed of gogo anime(Which is usually slow)
	* It requires chrome browser to be preinstalled 
	* you need to download the chrome driver and place it in the same directory of the program


In future I will add support to Other websites and add option to start the downloads from any episode
